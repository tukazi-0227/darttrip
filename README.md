Darttrip
====

Overview
自分が行きたい地方を選択しその中からランダムに一つの県を返してくれる旅行アプリです

## Description ##
#### ホーム画面 ####
  機能未実施
#### ダーツ画面 ####
  指定した地方から出発するボタンでランダムな県を返す
## Requirement ##
###### A5:SQL mk-2(My SQL)に接続する必要あり ######
###### next.js環境構築 ######
###### Spring boot環境構築 ######
## Install ##
#### フロントエンド ####
###### cd frontend ######
###### npm ci ######
###### npm run dev ######
